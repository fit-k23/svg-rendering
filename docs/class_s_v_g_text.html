Detailed Description

Class that represents Text object in svg.

Parameters
textAnchorAll text object has a text-anchor (default to start)
dxshift along Ox (default to be 0)
dyshift along Oy (default to be 0)
Note
dx, dy will be added to position so there are no attributes dx, dy
rotation is set to 0 by default (no rotation)
font size is default to be 1
