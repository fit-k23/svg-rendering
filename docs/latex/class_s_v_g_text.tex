\doxysubsection{Detailed Description}
Text Class. 

\begin{DoxyNote}{Note}
All text object has a text-\/anchor (default to start) 

dx, dy (shift along Ox, Oy coordinates) is default to be zero 

dx, dy will be added to position so there are no attributes dx, dy 

rotation is set to 0 by default (no rotation) 

font size default to be 1 
\end{DoxyNote}
